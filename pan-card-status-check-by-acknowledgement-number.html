PAN Card Status Check by Acknowledgement Number

Applicants can check their PAN card application status online using the acknowledgement number provided at the time of PAN application.

How to Check PAN Card Status Online

  1. Open the official PAN status website
  2. Select application type
  3. Enter acknowledgement number
  4. Submit to view PAN card status

Official PAN Card Status Check Links

  • Check PAN Card Status – UTI Portal
  • Check PAN Card Status – NSDL Portal

Details Required to Check PAN Status

  • Acknowledgement Number
  • Date of Birth (if asked)

Important Note

This page provides only official PAN card tracking links. No personal data is collected on this website.
